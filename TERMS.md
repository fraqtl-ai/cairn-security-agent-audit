# Evaluation Terms

This package is provided for design-partner evaluation of CAIRN Security Agent
Audit.

Permitted use:

- run locally on internal security-agent traces,
- inspect generated reports,
- share summary results with fraQtl for pilot scoping.

Not permitted without written permission:

- redistribution,
- resale,
- public hosting,
- production deployment,
- removal of notices,
- use as an automated pentest runner.

This package is audit-only. It does not execute pentest actions and does not
serve cached outputs.
